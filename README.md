# <img src="assets/logo.png" height="35" valign="middle"> Job Hunter AI

**Job Hunter AI** — это полностью бесплатный личный ассистент, который избавит вас от самой нудной рутины при поиске работы. Больше не нужно часами читать тонны «воды» в описаниях вакансий и вымучивать уникальные отклики для каждого работодателя. Нейросеть сделает это за вас.

Всё работает максимально просто: вы листаете вакансии в Chrome, нажимаете одну кнопку, а программа берет всю мысленную работу на себя.

### 🛠️ Технологический стек / Tech Stack

<details>
<summary><b>Развернуть детализацию стека (Tech Stack Details)</b></summary>

* **GUI & OS:** `customtkinter` • `Pillow` • `ctypes` (идеальный рендеринг шрифтов DPI-Awareness 2K/4K, принудительный темный заголовок Windows, кросс-раскладочные горячие клавиши).
* **Local API:** `Flask` • `flask-cors` • `threading` + потокобезопасная `queue.Queue` (асинхронный прием вебхуков из браузера и мгновенная очистка зависших портов).
* **AI Cascade:** Multi-LLM движок (**Gemini 3.1/3.5**, **OpenAI GPT-5/o3**, **Claude 4**, **DeepSeek**) • Отказоустойчивый каскад (Failover Chain) • Экспоненциальный откат (Exponential Backoff) • Умный парсер JSON.
* **Storage:** Легковесная локальная база на JSON-файлах в защищенной системной папке `%APPDATA%` • Циклическая автоочистка логов отклонений (строгий лимит в 50 записей).
* **DevOps & Build:** `PyInstaller` • Уникальный скрипт предсборочной подготовки `Self-Healing Refactor` (автоматическое устранение конфликтов маскирования библиотек и перелинковка импортов).

</details>

## 🚀 История обновлений (Changelog)

<details>
<summary><b>📦 v1.2.0 (Текущая версия)</b></summary>

* **[Engine]** Реализована мультимодельная архитектура на базе `BaseProvider` (Gemini, OpenAI, Anthropic, DeepSeek) с интуитивным пультом управления в главном окне.
* **[Engine]** Улучшена функция точечной генерации вакансиеориентированных сопроводительных писем под стек и требования работодателя в один клик.
* **[Engine]** Добавлен каскадный обработчик ошибок API (Failover Chain) и автоматическое исправление кривого JSON от LLM.
* **[UI]** Новый современный темный интерфейс на CustomTkinter с полной поддержкой High-DPI (шрифты и элементы идеально четкие на 2K/4K экранах).
* **[UI]** Полный отказ от `grab_set()`, вызывавшего дедлоки интерфейса; инкрементальный рендеринг списков вакансий.
* **[API]** Интегрирован настраиваемый таймаут-менеджер очереди (по умолчанию 15 секунд) со статус-баром обратного отсчета для защиты API-ключей и аккаунтов от блокировок.
* **[API]** Системный киллер зомби-процессов на порту 5000 для стабильного перезапуска локального Flask-сервера.
* **[Build]** Переработана структура проекта (`src/`), добавлен self-healing модуль для автоматического рефакторинга и сборки через PyInstaller.

</details>
