# Job Hunter AI by eric-esoteric

🎯 Job Hunter AI

Job Hunter AI — это ваш персональный интеллектуальный ассистент, который превращает хаотичный поиск работы в упорядоченный автоматизированный конвейер. Приложение работает в бесшовной связке со специальным браузерным расширением для Google Chrome: вы заходите на страницу любой вакансии, нажимаете одну кнопку, а программа мгновенно анализирует требования, сопоставляет их с вашим резюме, отсеивает сомнительные предложения и сразу пишет готовое сопроводительное письмо.

Больше не нужно тратить часы на чтение бесконечной "воды" в описаниях и вручную придумывать уникальные отклики для каждого работодателя. Нейросеть Gemini сделает это за вас за секунды!

Контролирует рабочую нагрузку: ИИ автоматически рассчитывает общее количество рабочих часов в неделю по графику. Если нагрузка превышает 45 часов (например, графики 5/2 по 10 часов или жесткие неоплачиваемые овертаймы) — вакансия будет отклонена.

Интеллектуальный контроль локации: По вашему желанию программа автоматически заблокирует вакансии, требующие обязательного физического присутствия в РФ или запрещающие использование корпоративного VPN из-за границы.

Удобно организует результаты: Все одобренные вакансии сохраняются в локальную базу данных. Вы можете управлять ими через стильный темный интерфейс (на базе CustomTkinter), просматривать детали, сравнивать условия и копировать сопроводительные письма в буфер обмена в один клик.

Ниже инструкция по установки:

⚙️ Инструкция по установке и запуску (для новичков)

Я постарался сделать процесс установки максимально простым. Просто следуйте шагам ниже.

Шаг 1. Установка программы на компьютер

Скачайте установочный файл JobHunterAI_Setup.exe из раздела релизов или соберите его самостоятельно.

Запустите установщик, ознакомьтесь и согласитесь с Лицензионным соглашением.

Дождитесь окончания копирования файлов.

В самом конце установки автоматически откроется папка extension (она понадобится нам для браузера на следующем шаге). Не закрывайте её.

Шаг 2. Установка расширения в Google Chrome

Наше расширение отправляет данные со страниц вакансий прямо в программу. Установить его очень легко:

Откройте браузер Google Chrome.

Вставьте в адресную строку ссылку chrome://extensions/ и нажмите Enter (вы попадете на страницу управления расширениями).

В правом верхнем углу экрана найдите и включите переключатель "Режим разработчика" (Developer mode).

В левом верхнем углу нажмите появившуюся кнопку "Загрузить распакованное расширение" (Load unpacked).

В открывшемся окне выбора папки укажите путь к папке extension (которая открылась на Шаге 1) и нажмите "Выбор папки".

На панели инструментов Chrome нажмите на иконку пазла (Расширения) и закрепите наше расширение булавкой, чтобы его иконка всегда была под рукой.

Шаг 3. Получение бесплатного ключа нейросети (Gemini API Key)

Приложение использует бесплатный искусственный интеллект от Google. Получить ключ можно буквально за 1 минуту:

Перейдите на сайт Google AI Studio и авторизуйтесь под своим обычным Google-аккаунтом.

В левом верхнем углу экрана нажмите синюю кнопку "Get API key".

Нажмите кнопку "Create API key", а затем выберите "Create API key in new project".

Скопируйте длинный сгенерированный код (он начинается на AIzaSy... или AQ.).

Шаг 4. Первый запуск и использование

Запустите программу Job Hunter AI с ярлыка на Рабочем столе.

Заполните личные данные:

Имя и Фамилия (будут использоваться для автографа в сопроводительных письмах).

Опыт работы и навыки (подробно опишите, что вы умеете делать — на основе этих данных ИИ будет оценивать вакансии и писать персонализированные письма).

Лицензия и ИИ-доступ (вставьте скопированный на Шаге 3 API-ключ Gemini).

Настройте галочки фильтрации работы (Удаленка, Офис, Гибрид, Без привязки к РФ) под ваши предпочтения.

Нажмите большую зеленую кнопку "ЗАПУСТИТЬ АССИСТЕНТА". Программа свернется в режим ожидания и начнет принимать данные.

Теперь откройте в браузере любую вакансию и просто нажмите на иконку нашего расширения на панели инструментов браузера. Иконка загорится оранжевым (...), а через пару секунд покажет статус OK (вакансия ушла на анализ) или ERR (если программа-помощник выключена).

Вся история одобренных вакансий и готовых писем будет бережно сохраняться в разделе "Открыть одобренные вакансии (Отобрано)"!
