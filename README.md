Job Hunter AI — это ваш персональный интеллектуальный ассистент, который превращает хаотичный поиск работы в упорядоченный автоматизированный конвейер. Приложение работает в бесшовной связке со специальным расширением для Google Chrome: вы заходите на страницу любой вакансии, нажимаете одну кнопку, а программа мгновенно анализирует требования, сопоставляет их с вашим резюме, отсеивает сомнительные предложения и пишет персонализированное сопроводительное письмо.

Больше не нужно тратить часы на чтение бесконечной "воды" в описаниях и вручную придумывать уникальные отклики для каждого работодателя. Нейросеть Gemini сделает это за вас за секунды!

Интуитивный темный интерфейс на базе CustomTkinter, адаптированный под любые High-DPI (2K/4K) мониторы.

✨ Ключевые возможности (Features)

⚡ Мгновенный перехват вакансий «на лету»: Браузерное расширение в один клик извлекает весь текст со страницы вакансии в Chrome и мгновенно передает его локальному приложению через защищенный CORS-совместимый вебхук.

⏳ Интеллектуальная очередь и таймер защиты API: Больше никаких банов API за частые запросы! При отправке пачки вакансий приложение выстраивает их в строгую очередь и обрабатывает поочередно, запуская 15-секундный таймер безопасности перед каждым ИИ-анализом.


Динамический статус-бар показывает обратный отсчет таймера и текущее количество задач в очереди.

🛡️ Двухстадийный ИИ-фильтр адекватности (Баланс 50/50): На первом этапе ИИ безжалостно отсеивает сомнительные вакансии (сетевой маркетинг, крипто-схемы, эзотерику, инфобизнес и «воду» без конкретных обязанностей), одобряя только качественные предложения.

🕒 Контроль рабочей нагрузки: ИИ автоматически рассчитывает суммарную недельную нагрузку по графику. Если график рабский (более 45 часов в неделю, неоплачиваемые переработки), вакансия мгновенно отправляется в отказ.

🌍 Гео-контроль и требования к VPN: Приложение автоматически заблокирует вакансии, требующие физического нахождения на территории РФ или содержащие жесткие ограничения на работу через удаленный VPN из-за границы.

📋 Раздельное управление результатами: Одобренные и отклоненные вакансии сортируются по удобным вкладкам. Детали отказов сохраняются в самоочищающийся журнал (до 50 записей).

⚙️ Инструкция по установке и запуску (для новичков)

Процесс развертывания проекта максимально прост и разделен на 4 шага.

Шаг 1. Установка программы на компьютер

Скачайте установочный файл JobHunterAI_Setup.exe из раздела Releases или соберите его самостоятельно с помощью скрипта build_exe.py.

Запустите установщик, ознакомьтесь и согласитесь с лицензионным соглашением.

Дождитесь окончания копирования файлов.

В конце установки автоматически откроется папка extension (она понадобится для браузера на следующем шаге). Не закрывайте её.

Шаг 2. Установка расширения в Google Chrome

Наше расширение парсит данные со страниц вакансий и передает их напрямую в программу:

Откройте браузер Google Chrome.

Перейдите по ссылке chrome://extensions/ (или зайдите в Меню -> Инструменты -> Расширения).

В правом верхнем углу включите тумблер «Режим разработчика» (Developer mode).

В левом верхнем углу нажмите кнопку «Загрузить распакованное расширение» (Load unpacked).

В открывшемся окне укажите путь к папке extension, которая открылась на Шаге 1, и нажмите «Выбор папки».

На панели инструментов Chrome нажмите на иконку пазла и закрепите расширение Job Hunter AI Parser булавкой.

Шаг 3. Получение бесплатного ключа нейросети (Gemini API Key)

⚠️ Важное примечание для пользователей из РФ (про VPN):

Так как проект использует Gemini API от Google, для работы приложения и получения ключа с территории РФ понадобится включенный VPN.

Почему именно Google Gemini? На сегодняшний день это единственная топовая нейросеть, которая выдает абсолютно бесплатные ключи API обычным пользователям. Согласитесь, один раз включить привычный VPN гораздо проще и быстрее, чем обычному человеку возиться с зарубежными картами, криптой и оплатой серверов условного платного DeepSeek или OpenAI.

Приложение использует официальный искусственный интеллект от Google. Получить ключ можно бесплатно за 1 минуту:

Перейдите на сайт Google AI Studio и авторизуйтесь под своим Google-аккаунтом.

В левом верхнем углу экрана нажмите синюю кнопку "Get API key".

Нажмите кнопку "Create API key", а затем выберите "Create API key in new project".

Скопируйте длинный сгенерированный код (он начинается на AIzaSy...).

Шаг 4. Первый запуск и использование

Запустите программу Job Hunter AI с ярлыка на Рабочем столе.

Заполните данные вашего профиля:

Имя и Фамилия (будут автоматически подставляться в подпись к сопроводительным письмам).

Ваш опыт работы и навыки (подробно опишите стек технологий и опыт — именно по этим данным ИИ будет фильтровать вакансии и генерировать отклики).

Лицензия и ИИ-доступ (вставьте полученный на Шаге 3 ключ API).

Настройте желаемые форматы работы (галочки: Удаленка, Офис, Гибрид, Без привязки к РФ).

Нажмите большую зеленую кнопку «ЗАПУСТИТЬ АССИСТЕНТА». Статус сменится на «Ожидание вакансий...».

Теперь откройте в браузере страницу любой вакансии и нажмите на иконку расширения.

Иконка загорится оранжевым (...), встанет в очередь, а по завершении анализа покажет статус OK (вакансия обработана) или ERR (если программа выключена или ключ недействителен).

Вся история и готовые письма будут бережно сохранены в разделе «Открыть одобренные вакансии (Отобрано)»!

🛠️ Технологический стек (Stack)

GUI-интерфейс: Python, customtkinter, tkinter.

Локальный API: Flask, flask-cors (для безопасных кросс-доменных запросов).

Сетевой движок: Прямые HTTP-запросы через urllib.request (защита от кодировочных сбоев Windows).

ИИ-движок: Google Gemini API (gemini-3.1-flash и резервные модели).

База данных: Локальные JSON-хранилища со встроенной автоочисткой логов.

Сборщик: PyInstaller (автоматическая линковка тем CustomTkinter и ассетов).
